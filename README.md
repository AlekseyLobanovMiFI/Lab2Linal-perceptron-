# 1. Теоретическая часть

Однослойный перцептрон — это простейшая нейронная сеть для решения задач бинарной классификации.

Модель вычисляет линейную комбинацию признаков:

\[
z = w^Tx + b
\]

где:

- \(w\) — вектор весов;
- \(x\) — входные признаки;
- \(b\) — смещение.

Для получения вероятности принадлежности к классу используется сигмоидная функция активации:

\[
\sigma(z)=\frac{1}{1+e^{-z}}
\]

Функция потерь — бинарная кросс-энтропия:

\[
L=-\frac{1}{N}\sum_{i=1}^{N}\left[y_i\log(\hat y_i)+(1-y_i)\log(1-\hat y_i)\right]
\]

Обучение выполняется методом градиентного спуска с использованием mini-batch.

# 7. Эксперимент: влияние инициализации весов

Сравнивались:

- нулевая инициализация;
- маленькие случайные веса;
- большие случайные веса.

## Выводы

Нулевая инициализация ухудшает обучение, так как веса обновляются одинаково.

Большие случайные веса приводят к насыщению sigmoid-функции:

\[
\sigma(z) \approx 0 \; \text{или} \; 1
\]

что вызывает исчезновение градиентов.

Наилучшие результаты показала инициализация маленькими случайными значениями.

# 8. Кастомные наборы данных

Были реализованы дополнительные генераторы данных:

- линейно разделимые данные;
- XOR;
- окружность.

## XOR

Для XOR-подобного набора данных были получены результаты:

| Метрика | Значение |
|---|---|
| Accuracy | ≈ 0.52 |
| Loss | ≈ 0.69 |

Модель не смогла обучиться.

## Причина

Однослойный перцептрон способен строить только линейную разделяющую границу.

XOR не является линейно разделимой задачей.

Следовательно, одной прямой невозможно разделить классы.

Это демонстрирует ограниченность однослойного перцептрона.
