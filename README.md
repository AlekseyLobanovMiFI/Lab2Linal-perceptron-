# Теоретическая часть

Линейная модель:

```math
z = w^Tx + b
```

Сигмоидная функция:

```math
\sigma(z)=\frac{1}{1+e^{-z}}
```

Функция потерь Binary Cross-Entropy:

```math
L=-\frac{1}{N}\sum_{i=1}^{N}\left[y_i\log(\hat y_i)+(1-y_i)\log(1-\hat y_i)\right]
```

# Эксперимент: XOR

Для XOR-подобного набора данных модель не смогла обучиться.

Полученные результаты:

| Метрика | Значение |
|---|---|
| Accuracy | ≈ 0.52 |
| Loss | ≈ 0.69 |

## Вывод

Однослойный перцептрон способен строить только линейную разделяющую границу.

XOR не является линейно разделимой задачей,
поэтому модель не смогла корректно разделить классы.

Это демонстрирует ограниченность однослойного перцептрона.

---

# Эксперименты с learning rate

Проверялись значения:

- 0.001
- 0.01
- 0.5
- 1.0

## Выводы

- Маленький learning rate приводит к очень медленному обучению.
- Большой learning rate вызывает колебания функции потерь.
- Значения 0.1–0.5 показали наиболее стабильное обучение.

---

# Эксперименты с batch size

Проверялись значения:

- 1
- 16
- 64
- 256

## Выводы

- Маленький batch size делает обучение шумным.
- Большой batch size делает обучение более стабильным.
- Batch size = 32 или 64 показал лучшие результаты.

---

# Эксперимент: инициализация весов

Сравнивались:

- нулевая инициализация;
- маленькие случайные веса;
- большие случайные веса.

## Выводы

Большие случайные веса приводят к насыщению sigmoid-функции,
из-за чего ухудшается обучение модели.

Нулевая инициализация также показала плохие результаты.

Наилучшее качество показала инициализация маленькими случайными значениями.
